Implement a method for finding min and max elements of a stream in accordance with a given comparator. The stream elements are of generic type T, so you should use the given Comparator<? super T> order in order to find the minimum and maximum.

After you find the minimum and maximum elements, you should pass them to minMaxConsumer in the following way:

minMaxConsumer.accept(min, max);

If the stream doesn't contain any elements, invoke:

minMaxConsumer.accept(null, null);

Remember, that as soon as you call any terminal operation, the stream is closed and you can get an IllegalStateException.



Memory limit: 256 MB
Time limit: 5 seconds
